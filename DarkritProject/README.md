# DarkritProject

Simple 2D Engine to make games. Development started recently so there won't be tests nor documents. Chances are I end up also adding game assets here while I develop the engine. Ideally this repo should only be the Engine Wrapper, but untils things are ready I'll allow myself some flexibility.

## List of features
- Simple hot reloading systemç
- Simple, performant and limited ECS
- Simple InstancedQuadRenderer, really limited at the moment
- Input system
	- Allows to define multiple bindings per action
	- [NOT IMPLEMENTED YET] Allows to record and replay input. It assumes fixed tick simulation for multiplayer determinism
